# mvc-high-street-gym-app

A full-stack gym management system developed using modern web technologies.

## Technologies Used

### Backend

- Node.js
- Express.js
- MVC Architecture
- EJS Template Engine
- MySQL Database
- RESTful API
- Authentication & Authorization

### Frontend

- React.js
- React Router
- Tailwind CSS
- React Icons
- Fetch API

## Features

- User authentication (Login & Registration)
- Role-based access (Admin, Trainer, Member)
- Session management
- Activity management
- Booking system
- Location management
- Search and filtering
- Responsive design
- REST API integration
- Server-side rendering using EJS
- Client-side application using React
